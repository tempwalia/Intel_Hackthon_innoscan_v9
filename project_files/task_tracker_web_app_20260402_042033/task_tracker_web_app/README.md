# Task Tracker Web App  

## Overview  
A simple web application to add, view, and delete daily tasks using HTML, CSS, and JavaScript. Tasks are persisted in browser local storage.  

## Features  
- Add tasks with optional due dates  
- View all tasks in a list  
- Delete individual tasks  
- Persistent storage using local storage  

## Setup Instructions  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/your-username/task_tracker_web_app.git  
   ```  
2. Navigate to the project directory:  
   ```bash  
   cd task_tracker_web_app  
   ```  
3. Open `index.html` in your browser to use the app  

## Run Instructions  
- No server required (single-page app)  
- Open `index.html` directly in a browser
